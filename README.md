# cid11
Buscador de CID-11 desenvolvido para facilitar a pesquisa e identificação de doenças, transtornos e condições de saúde de forma rápida e prática. Permite localizar códigos oficiais da Classificação Internacional de Doenças (CID-11), auxiliando profissionais da saúde, estudantes e instituições no diagnóstico, documentação e organização clínica.
